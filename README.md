## About

I'm a software engineer with a BSc in Computer Science from Haramaya University. I specialize in full-stack web development — Django and Node.js on the backend, React and Next.js on the front — with a particular interest in modernizing aging PHP/Django codebases and building locally relevant tools for the Ethiopian context.

I care about writing software that actually gets used. That means clean APIs, sensible database schemas, and UIs that don't confuse people.

- 🔭 Currently working on **Node.js microservices and reusable React component libraries**
- 🌱 Going deeper on **SQL and NoSQL query optimization**
- 🤝 Open to **freelance work and meaningful collaborations**
